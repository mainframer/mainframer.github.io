Linux压缩命令

在平常的Linux接触过程中，各种压缩文件经常遇到。使用压缩文件的好处显而易见，可以节省空间并且节约网络流量传输的开销，所以学会各种压缩文件的使用是不可避免的一项技能。自己平常使用的再加上一些网上搜集的，估计够全了，简单地写在这记下，方便查阅： 压缩文件 (解释: x——解压 c——压缩 v——显示过程 f——压缩包名字)
tar格式

tar xvf A.tar
tar cv A.tar directory

gz格式

gunzip -d A.gz
也可用 gzip——gzip A.gz

tar.gz/tgz格式

tar zxvf A.tar.gz 解压至当前目录
tar zcvf A.tar.gz orderFolder 压缩orderFoler里面的文件至A.tar.gz

bz2格式

bzip2 -d A.bz2
bzip2 -z A.bz2

tar.bz2格式

tar jxvf A.tar.bz2
tar jcvf A.tar.bz2 orderFolder 压缩orderFoler里面的文件至A.tar.bz2

bz格式

bzip2 -d A.bz

tar.bz格式

tar jxvf A.tar.bz

tar.z格式

tar Zxvf A.tar.z tar
Zcvf A.tar.Z Folder

zip格式

unzip file.zip
zip A.zip folder

rar格式

rar a A.rar
rar e A.rar

deb格式

arp A.deb

bundle类型安装

sudo chmod +x A.bundle  sudo ./A.bundle
