ISPF-提高效率少用PF3和START

同一个PDS下，记得多用BROWSE, VIEW,EDIT,COMP这些命令。比如BROWSE TEST02 或者EDIT TEST03 或者VIEW TEST04 或者COMP 'IBMUSER.PROJ.TEST.JCL1((TEST01)' 这样，不需要另外PF3或者START开一个窗口。 比如需要在同一个PDS下的TEST02这个JCL中拷贝一些东西到当前的TEST01的时候，经常用到。

File  Edit  Edit_Settings  Menu  Utilities  Compilers  Test  Help            
 ```````````````````````````````````````````````````````````````````````````````
 VIEW       IBMUSER.PROJ.TEST.JCL(TEST01) - 01.10           Columns 00001 00072 
 Command ===> BROWSE TEST02                                    Scroll ===> CSR  
 ****** ***************************** Top of Data ******************************
 000001 //IBMUSERS JOB                      011000010TEST,                      
 000002 //  CLASS=1,MSGCLASS=R,NOTIFY=&SYSUID                                   
 000003 //STEP01 EXEC PGM=TESTJOB                                               
 000004 //STEPLIB  DD   DISP=SHR,DSN=IBMUSER.LBN.LOAD.LIB                       
 000005 //FINPUT   DD DSN=IBMUSER.TEMPFLIN,DISP=SHR                             
 000006 //FOUTPUT  DD DSN=IBMUSER.TEMPFLOUT,DISP=SHR                            
 000007 //SYSPRINT DD  SYSOUT=*                                                 
 000008 //SYSOUT   DD  SYSOUT=*                                                 
 000009 //SYSUDUMP DD  SYSOUT=R                                                 
 000010 //PLIDUMP  DD  SYSOUT=R

如果记不得member的名字，可以只输入BROWSE或者EDIT或者VIEW，让系统列出PDS下所有member供你pick up。
