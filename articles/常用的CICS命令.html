常用的CICS命令

2012-03-01

我们可以在COBOL，PL/I，Assembler等应用程序中写CICS命令，在COBOL和PL/I中，格式分别如下所示：

COBOL

{% codeblock lang:ruby %} EXEC CICS function[option[(arg)]]... END-EXEC

PL/I

{% codeblock lang:ruby %} EXEC CICS function[option[(arg)]]... ; (末尾有分号) {% endcodeblock %}
CICS的命令很多，不过最常用的也就是如下几个。所以，只要熟悉这几个的用法就好了，其他的当做了解即可，需要的时候再去查 IBM CICS command summary或者CICS手册。 {% codeblock lang:ruby %} EXEC CICS ABEND [ABCODE(arg)] EXEC CICS ASKTIME [ABSTIME(arg)] EXEC CICS ASSIGN [OPID(arg)] EXEC CICS RETURN [TRANSID(arg)] EXEC CICS HANDLE CONDITION EXEC CICS IGNORE CONDITION EXEC CICS READQ TS [QUEUE(arg)] EXEC CICS WRITEQ TS [QUEUE(arg)] EXEC CICS DELETEQ TS [QUEUE(arg)] EXEC CICS RECEIVE MAP EXEC CICS SEND MAP EXEC CICS XCTL PROGRAM EXEC CICS LINK PROGRAM
