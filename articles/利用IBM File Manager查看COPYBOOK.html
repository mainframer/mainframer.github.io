利用IBM File Manager查看COPYBOOK

这篇文章就来说下在大机上如何利用IBM File Manager来查看copybook的结构，特别是每个字段的起始位置。

1.在Command栏输入0;5;1 设置自动检测语言

这样，不论你的copybook是PLI的还是COBOL的，FileManager都能自动检测出来。

2.设置好之后，进入3；13 (Utility-&gt;Copybook)
