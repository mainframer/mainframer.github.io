'ISPF-只显示含关键字的所有行'

2012-09-07

命令： {% blockquote %} X ALL; FIND keyword ALL; {% endblockquote %} 比如，只显示所有含DT0907的行，可以这么做，先X ALL;再FIND DT0907 ALL; {% codeblock %} VIEW IBMUSER.TESTJCL(TESTJOB) - 01.9 4 CHARS 'DT0907' Command ===> find dt0907 all Scroll ===> CSR
