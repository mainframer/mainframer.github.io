PL/I编译器版本

2012-02-02

PL/I编译器的版本，如下所示： {% codeblock %} Product Product name   Compiler "signature" Module prefix 360S-NL-511    S/360 OS PL/1 (F)             '   '                  IEM,IHE,IHL 5734-PL1       OS PL/I OPTIMIZING COMPILER V1   5734-PL1           IEL,IBM 5734-PL2       OS PL/I CHECKOUT COMPILER 5734-PL3       OS PL/I Opt Comp Lib Kanji Name Support 5668-909       OS PL/I V2 (COMP/LIB/TEST) 5668-910       OS PL/I V2 (COMP/LIB)                 5668-910              IEL,IBM 5668-911       OS PL/I V2 (Library) IBM 5688-235       PL/I for MVS AND VM V1               5688-235              IEL,IBM 5655-B22       VisualAge PL/I for OS/390 V2       5655-B22             IEL,IBM 5655-H31       Enterprise PL/I for z/OS V3         5655-H31             IBM {% endcodeblock %} 这里是原链接地址：http://gsf-soft.com/Documents/PRODCODES.shtml
编译完PL/I程序之后，在生成的loadmodule二进制文件里面查找Compiler signature(如5655-235),对照上表即可知道编译器的版本。
