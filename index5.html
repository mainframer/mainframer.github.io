SOME INPUT DATA IGNORE
2012-05-03

在编辑dataset的时候如果右上角提示SOME INPUT DATA IGNORE,请记得把每行末尾的行号关闭: {% blockquote %}
NUM OFF {% endblockquote %}
否则，后面8位就是行号，将此文件作为程序的input的时候就危险了!

S000 U0004 ABEND ERROR
2012-04-30

遇到了'S000 U0004 ABEND ERROR'错误。
一如既往地分析dump，发现是由于我在JCL里面添加了'PARM='TEST(ALL,,,VTAM%IBMUSER:)'的debug参数，而我要debug的程序的loadmodule不在test环境里面，导致JCL试图去debug生产环境的loadmodule，抛出'S000 U0004‘错误，应该是access vialation之类的错误吧,修改完loadmodule的路径就好了。

PLI打印出PIC类型输出
2012-04-01

使用PLI打印输出信息，最常见的是: {% codeblock lang:python %} PUT SKIP(1) EDIT ('CODE = ',ERR_CODE)(A,A); {% endcodeblock %} 这里是打印PIC类型的输出，不怎么常用。今天用到了，所以记在这:
{% codeblock lang:python %} PUT EDIT('ACCOUNT PROCESSED : ',ACCOUNT_CTR)
(SKIP(1),COL(1),A,P'999999999');

PUT SKIP FILE(OUTFILE) EDIT('CHENGDU: ',SOURCE) (A,P'ZZ,ZZZV.ZZ');
