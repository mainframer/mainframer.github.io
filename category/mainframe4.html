ISPF中Shift+PageUp和UNDO的区别

Date 2012-05-15 By mainframer Category Mainframe. Tags Shift+PageUp

两者都是撤销，相当于windows下的'Ctrl+z'。但分工不同，Shift+PageUp只能撤销未按回车之前的修改动作，对于按了回车就无力回天了。
而UNDO负责的是撤销回车之后的修改动作，请注意务必要事先用命令'Rec on'打开ISPF的recovery开关。

Date 2012-04-30 By mainframer Category Mainframe. Tags S000 U0004

PLI打印出PIC类型输出

Date 2012-04-01 By mainframer Category Mainframe. Tags PLI PIC

使用PLI打印输出信息，最常见的是:

PUT SKIP(1) EDIT ('CODE = ',ERR_CODE)(A,A);

这里是打印PIC类型的输出，不怎么常用。今天用到了，所以记在这: 

PUT EDIT('ACCOUNT PROCESSED : ',ACCOUNT_CTR)      
                  (SKIP(1),COL(1),A,P'999999999'); 

PUT SKIP FILE(OUTFILE) EDIT('CHENGDU: ',SOURCE) 
      (A,P'ZZ,ZZZV.ZZ');
